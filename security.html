Trust & Safety

Security at CloseDraft

We take the security of your data seriously. Here's how we protect your information.

Encryption

All data is encrypted in transit using TLS 1.3 and at rest using AES-256 encryption.

Row Level Security

Supabase RLS ensures you can only access your own data. Every query is scoped to your user ID.

Secure Authentication

Google OAuth and Supabase Auth provide industry-standard authentication with optional 2FA.

Infrastructure Security

Hosting & Network

  • ✓ Hosted on Render with DDoS protection
  • ✓ Cloudflare for DNS and CDN
  • ✓ All traffic encrypted via HTTPS
  • ✓ Regular security patches and updates

Database Security

  • ✓ Supabase PostgreSQL with RLS
  • ✓ Automated daily encrypted backups
  • ✓ Point-in-time recovery enabled
  • ✓ Connection pooling and SSL required

API Security

  • ✓ Bearer token authentication on all endpoints
  • ✓ Rate limiting on authentication endpoints
  • ✓ Input validation and sanitization
  • ✓ CORS restricted to our domains

Payment Security

  • ✓ PCI-compliant via Polar.sh
  • ✓ We never store full credit card numbers
  • ✓ 3D Secure authentication supported
  • ✓ Fraud detection and prevention

AI & Data Privacy

Google Gemini AI

We send only the necessary context (client name, business, project) to generate emails. Google does not use your data to train their models. Learn more →

Your Data is Yours

We never sell or share your client data. You can export or delete your data at any time from Settings.

Report a Security Issue

We take security reports seriously. If you believe you've found a security vulnerability in CloseDraft, please email us at:

security@closedraft.com

We'll respond within 48 hours. Please do not disclose vulnerabilities publicly until we've had a chance to address them.
